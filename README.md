# DesktopClock
一个简约简单的，由WPF构建的，桌面时钟（默认适配4K分辨率，建议至少2.5K分辨率） 

- 根据系统明暗主题，自动调节配色（建议配合Auto Dark Mode食用）
- 支持自动定位，调用API显示天气信息（默认一小时刷新一次，需要安装有支持Emoji的字体（Win10自带））.
- 支持天气文字自动变色，天气图标、气温和风速将自动变色。
- 同时也支持随时手动刷新。
- 按住可拖动位置，拖动后的位置将自动保存，下次打开仍然会在原先位置
